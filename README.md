# Data-Pipeline-Web-Application-Project
# Project Overview
an end-to-end data pipeline and web application using Python (pandas), MySQL, and Node.js to analyze EV adoption trends.
## Tech Stack

* Python (pandas)
* MySQL
* Node.js (Express)
* Chart.js

## Architecture

Data → Cleaning → MySQL → Backend → Web Interface

## Key Features

* Data cleaning and transformation
* Normalized database schema (3NF)
* Backend API for querying data
* Interactive dashboard
